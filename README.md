# Gomoku
同济大学程序设计范式期中五子棋项目

# 主要的技术/优化方法
- AlphaBeta剪枝
- VCF/VCT计算（基于AlphaBeta剪枝）
- 迭代加深
- 启发式搜索
- Zobrist置换表
- 开局库
- 动态更新全局评估值（基于点）
- 采用哈希表快速估值
